🎓 CourseEnrollmentProject
An automated academic registration system for students and institutions.

CourseEnrollmentProject handles the complex logic of academic registration, including prerequisite checks, capacity management, and student schedules.

🚀 Key Features
🏫 Smart Enrollment: Automatic validation of course prerequisites and student eligibility.

📅 Schedule Builder: Generates conflict-free timetables for students and professors.

⚡ Real-time Capacity: Thread-safe handling of course registration to prevent over-enrollment.

📂 Grade Management: Modules for instructors to manage student performance and final transcripts.

🛠️ Tech Stack
Core: Java 21, Spring Boot 3.x

Persistence: PostgreSQL + Spring Data JPA

Migrations: Flyway

Utils: Lombok, MapStruct (for high-performance DTO mapping)
